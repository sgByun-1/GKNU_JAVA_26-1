# 시작
```java
public class C260313 { //소스파일: C260313.java

    public static void main(String[] args){ //main()메소드에서 실행 시작
        /*
        * class: 자바프로그램의 시작/ 데이터(필드)+동작(메서드) 묶음
        * main(): 실행의 출발점/ 실행 시작점, JVM(자바가상머신)의 동작: main탐색 후 호출/ 실행코드는 우선 main 내부에 배치
        * package:
        * import:

        * 식별자: 이름붙이기 개념
        * 프로그램 안의 대산에 붙이는 이름(클래스/변수/상수/메서드 이름 등)
        */
        System.out.println("Hello, World!"); //"Hello, World!" 문자열을 콘솔(화면)에 출력
                                             //c/cpp와 다르게 표준 입출력은 시스템이 out을 해줘야함. =>다른건 다 편한데 출력 입력이 조금 번거로움
    }
}
```
